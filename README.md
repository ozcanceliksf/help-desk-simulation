# Help Desk Simulation

A collection of realistic IT help desk scenarios with structured troubleshooting workflows, ticket documentation, and resolution notes.

Built to demonstrate practical IT support skills for entry-level Help Desk and Technical Support roles.

## Ticket Categories

- 🖥️ Hardware issues (monitor, keyboard, printer, laptop)
- 🌐 Network connectivity (no internet, slow connection, VPN)
- 🔐 Account & access (password reset, account lockout, MFA)
- 📧 Email & communication (Outlook, Teams, calendar sync)
- 💾 Software issues (installation, crashes, updates)
- 🖨️ Printer & peripheral setup

---

## Skills Demonstrated

- Ticket lifecycle management (open → in progress → resolved)
- Root cause analysis
- Clear technical documentation
- User communication and escalation procedures
- Following standard troubleshooting methodology

---

## Troubleshooting Methodology

Every ticket follows this structure:

1. **Identify** — Gather information from the user
2. **Reproduce** — Confirm the issue exists
3. **Isolate** — Narrow down the root cause
4. **Fix** — Apply the solution
5. **Verify** — Confirm resolution with the user
6. **Document** — Record steps taken for future reference
